# DeepQuest DAPP 产品说明

## 产品定位

**DeepQuest** 是一个基于 BSC（币安智能链）的 Web3 DeFi 量化平台，用户通过连接钱包地址直接参与，无需传统注册流程。

## 核心概念

### 代币信息

| 属性 | 值 |
|------|-----|
| 代币名称 | DQ (DeepQuest Token) |
| 合约链 | Solana |
| 兑换对 | DQ ↔ SOL |
| SOL 合约地址 | `0x570A5D26f7765Ecb712C0924E4De545B89fD43dF` |

### Web3 DAPP vs 传统 Web 应用

| 维度 | 传统 Web 应用 | DeepQuest DAPP |
|------|--------------|----------------|
| **身份标识** | 用户名/邮箱 | BSC 钱包地址（0x...） |
| **登录方式** | 密码登录 | 钱包签名授权 |
| **数据存储** | 数据库 | 链上合约 + 链下索引 |
| **资产管理** | 平台托管 | 用户自主掌控 |

### BSC 区块链标准

- **Chain ID**: 56 (主网) / 97 (测试网)
- **地址格式**: 以太坊格式（0x + 40位十六进制）
- **示例**: `0x7cH3JkLmNoPqRsTuVwXyZaBcDeFgHiJkLmNoPq`
- **区块浏览器**: https://bscscan.com

## 核心功能

### 1. 质押（Stake）

用户将 DQ 代币质押到智能合约，获取量化交易产生的收益分红。

**质押周期与收益：**

| 周期 | 分红比例 |
|------|----------|
| 30 天 | 5% |
| 90 天 | 10% |
| 180 天 | 15% |
| 360 天 | 20% |

**收益来源**：量化策略交易利润，每日自动计算，按期发放。

### 2. 兑换（Swap）

DQ 代币与 SOL 之间的双向兑换。

- **DQ**: DeepQuest 平台代币
- **兑换池**: 流动性支持
- **兑换比例**: 实时价格（通过 DEX 聚合器获取）

### 3. 节点 NFT 卡牌

用户使用 USDT 购买 NFT 节点卡牌，享受节点权益。

**卡牌类型：**

| 类型 | 价格 | 权益 |
|------|------|------|
| A 级节点卡 | 较高 | 高级权益 + 高比例分红 |
| B 级节点卡 | 中等 | 中级权益 + 中比例分红 |
| C 级节点卡 | 基础 | 基础权益 + 入门分红 |

**资金分配：**

| 用途 | 比例 |
|------|------|
| LP 质押池 | 60% |
| 节点 NFT | 15% |
| 运营基金 | 25% |

### 4. 推广奖励

基于推荐关系的奖励机制。

- **直推奖励**: 直接推荐人获得奖励
- **团队奖励**: 团队整体业绩奖励
- **邀请码**: 用户的钱包地址作为唯一邀请标识

## 用户旅程

```
1. 连接钱包 (MetaMask / Trust Wallet / 钱包应用)
         ↓
2. 领取 DQ 代币 (通过领取页面获得初始代币)
         ↓
3. 质押 DQ (选择质押周期：30/90/180/360天)
         ↓
4. 获取收益 (每日自动计算，按期发放)
         ↓
5. 兑换 DQ (在 DEX 中将 DQ 兑换为 SOL)
         ↓
6. 购买节点 (使用 USDT 购买节点 NFT 卡牌)
         ↓
7. 邀请好友 (复制邀请链接，分享推广)
         ↓
8. 团队奖励 (直推奖励 + 团队业绩奖励)
```

## API 接口

### 后端 API Base URL

```
${EXPO_PUBLIC_BACKEND_BASE_URL}/api/v1
```

### 主要接口

#### 用户数据

```
GET /api/v1/dapp/users/:address
```

#### 质押操作

```
GET  /api/v1/dapp/stakes/:address      # 获取质押记录
POST /api/v1/dapp/stakes               # 创建质押
```

#### 兑换

```
GET  /api/v1/dapp/swap/quote           # 获取兑换报价
POST /api/v1/dapp/swap/execute         # 执行兑换
```

#### 节点

```
GET  /api/v1/cards                      # 获取卡牌列表
GET  /api/v1/dapp/nodes/:address        # 获取用户节点
POST /api/v1/dapp/nodes                 # 购买节点
```

#### 团队

```
GET /api/v1/dapp/team/:address         # 获取团队数据
```
