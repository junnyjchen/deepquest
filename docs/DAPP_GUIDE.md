# DeepQuest DAPP 产品说明

## 产品定位

**DeepQuest** 是一个基于 BSC（币安智能链）的 Web3 DeFi 量化平台，用户通过连接钱包地址直接参与，无需传统注册流程。

## 核心概念

### Web3 DAPP vs 传统 Web 应用

| 维度 | 传统 Web 应用 | DeepQuest DAPP |
|------|--------------|----------------|
| **身份标识** | 用户名/邮箱 | BSC 钱包地址（0x...） |
| **登录方式** | 密码登录 | 钱包签名授权 |
| **数据存储** | 数据库 | 链上合约 + 链下索引 |
| **资产管理** | 平台托管 | 用户自主掌控 |

### BSC 区块链标准

- **Chain ID**: 56 (主网) / 97 (测试网)
- **地址格式**: 以太坊格式（0x + 40位十六进制）
- **示例**: `0x7cH3JkLmNoPqRsTuVwXyZaBcDeFgHiJkLmNoPq`
- **区块浏览器**: https://bscscan.com

## 核心功能

### 1. 质押（Stake）
用户将 BNB 质押到智能合约，获取量化交易产生的收益。
- **年化收益**: 12% - 36%
- **锁定期**: 30天起
- **收益来源**: 量化策略交易利润

### 2. 兑换（Swap）
DQT 代币与 BNB 之间的兑换。
- **DQT**: DeepQuest 平台代币
- **兑换池**: 流动性支持

### 3. 推广奖励
基于推荐关系的奖励机制。
- **直推奖励**: 直接推荐人获得奖励
- **团队奖励**: 团队整体业绩奖励
- **邀请码**: 用户的钱包地址作为唯一邀请标识

### 4. 节点机制
成为节点合伙人享受更多权益。
- **节点类型**: 节点合伙人、节点委托人
- **申请流程**: 提交申请 → 审核 → 激活

## 数据指标

### 平台数据
| 指标 | 说明 |
|------|------|
| 总质押量 | 平台累计接受的 BNB 质押总额 |
| 今日新增 | 当日新增的质押量 |
| DQT 价格 | 代币当前市场价格 |
| 累计发放奖励 | 平台累计分配的奖励总额 |

### 个人数据
| 指标 | 说明 |
|------|------|
| 我的质押 | 当前质押的 BNB 数量 |
| 我的收益 | 累计获得的 DQT 奖励 |
| 团队人数 | 直属团队成员数量 |
| 质押天数 | 当前质押的持续时间 |

## 用户旅程

```
1. 连接钱包 (MetaMask / Trust Wallet / 钱包应用)
         ↓
2. 激活账户 (支付小额激活费用)
         ↓
3. 质押 BNB (选择质押数量和期限)
         ↓
4. 获取收益 (每日自动计算，按期发放)
         ↓
5. 邀请好友 (复制邀请码，分享链接)
         ↓
6. 团队奖励 (直推奖励 + 团队业绩奖励)
```
